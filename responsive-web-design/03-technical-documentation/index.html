Introduccion

HTML y CSS son los lenguajes fundamentales para la construcción de páginas web.

HTML estructura el contenido, mientras que CSS se encarga de su presentación.

Dominar ambos lenguajes es esencial para cualquier desarrollador web.

Esta guía ofrece una visión general rápida para comenzar.

Aprenderás desde la estructura básica hasta conceptos de diseño responsive.

Sintaxis Basica

HTML utiliza etiquetas para definir elementos.

CSS aplica estilos a los elementos HTML mediante reglas.

Cada regla CSS tiene un selector y un bloque de declaraciones.

Las declaraciones incluyen una propiedad y un valor.

Veamos un ejemplo de código CSS básico:

p { color: blue; }
  • Etiquetas HTML van entre corchetes angulares
  • Las reglas CSS terminan con punto y coma
  • Los selectores indican a qué elemento aplicar el estilo
Selectores CSS

Los selectores determinan qué elementos se ven afectados por los estilos.

Existen varios tipos de selectores en CSS.

El selector universal aplica estilos a todos los elementos.

Los selectores de clase utilizan un punto antes del nombre.

Los selectores de id usan el símbolo "#".

* { margin: 0; }
.titulo { font-size: 2em; }
#principal { background: lightgray; }
  • Selector universal: *
  • Selector de clase: .clase
  • Selector de ID: #id
Responsive y buenas practicas

Un diseño responsive se adapta a diferentes tamaños de pantalla.

Se logra mediante media queries y diseño fluido.

Es importante mantener el código limpio y bien indentado.

Evita usar estilos en línea siempre que sea posible.

Utiliza herramientas como validadores de HTML y CSS para mejorar tu código.

@media (max-width: 768px) {
  body {
    background: white;
  }
}
  • Usa media queries para adaptar estilos
  • Aplica principios de diseño accesible
  • Organiza tu CSS en bloques lógicos
